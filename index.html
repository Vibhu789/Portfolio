Project 2

Optimized-Housing

The Optimized Housing is a Java-based project that demonstrates how to maximize comfort (utility score) within a limited housing space using algorithm optimization techniques.
This project compares: Brute Force Approach (Recursive), Dynamic Programming Approach (Optimized).
Problem Statement Given: A set of housing items/features Each item has: spaceRequired → space it occupies (in sq ft) utilityScore → comfort/benefit value A limited roomCapacity
Goal: Select items in such a way that: Total space does NOT exceed capacity Total utility score is MAXIMIZED Each item can either be: Included (1) Excluded (0) TC: O(2^n) -> O(n * Capacity) This makes it a 0/1 Knapsack problem. Link

Project 3

Autism Prediction

The AutismPrediction project is a machine learning–based system designed to predict the likelihood of Autism Spectrum Disorder (ASD) using behavioral, demographic, and questionnaire-based data.
Challenges: Handling imbalanced data, missing values, overfitting, and ensuring ethical, accurate autism prediction models.
Technology: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Data Preprocessing
Workflow : Data Collection → Data Loading → Exploratory Data Analysis → Data Cleaning → Data Preprocessing → Encoding → Feature Scaling → Class Balancing → Model Training → Model Validation → Performance Evaluation → Model Selection → Model Serialization → Deployment → Prediction Link
